# DiscDawg assets

Drop your logo and other brand assets here.

- **Logo:** Add `logo.svg` or `logo.png` — it’s used in the header and on the landing page hero. Prefer SVG for a crisp look at any size; PNG works too (e.g. 2x for retina).
- Other assets (favicon, product shots, etc.) can go in this folder and be referenced as `/assets/filename.ext`.
